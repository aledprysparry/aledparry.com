# UX Audit: GuessThePrice

> Step 1 of UPGRADE.md pipeline. Completed 2026-03-30.
> Auditor: Claude (UX Critic role)

---

## Critical Issues (P0 — Breaks Workflow)

1. **Score reset on round switch** — Host increments score during live show, clicks next round, score resets to saved value. Game-breaking for live use.
2. **Hero photo index bug on delete** — Deleting a non-hero photo doesn't update the hero index. Points to wrong photo after deletion.
3. **Export progress hangs** — MOV/ZIP exports take 10-30s with no progress bar or cancel button. Users think app froze.
4. **State sync race conditions** — Multiple debounced saves + episode switching can cause data loss.
5. **Episode switching without save prompt** — User loses unsaved changes silently.

## Major Issues (P1 — Usability)

6. **No input validation** — Address field accepts empty input. Empty rounds export placeholder text.
7. **Live mode touch targets too small** — Flow dots are 8px, +1 buttons are 40x20px. iPad users will miss.
8. **Photo manager drag feedback invisible** — No ghost image or drop animation. Reorder feels broken.
9. **Header button overload** — 9+ buttons in one row. Wraps on tablets.
10. **No keyboard navigation in live mode** — Touch only. Breaks iPad keyboard use.

## Minor Issues (P2 — Accessibility)

11. **No aria-labels on emoji icons** — Screen readers read raw emoji.
12. **Forms lack proper label elements** — `<div>Label</div> <input>` breaks form association.
13. **Status messages not announced** — "unsaved" and export status not in aria-live regions.
14. **Canvas has no fallback text** — No description for non-visual access.
15. **Colour-only status indicators** — Push-to-live dot relies solely on colour.

## Nits (P3 — Polish)

16. **No hover/focus states on buttons** — Can't tell if button is interactive until clicking.
17. **Sidebar scroll indicator missing** — User might not realise controls scroll.
18. **Export file naming opaque** — "gtp_property_r3.png" not descriptive.
19. **Photo manager modal has no entrance animation** — Appears instantly, feels jarring.
20. **Export doesn't restore canvas state** — Preview jumps to p=1 after export.
